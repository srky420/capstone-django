This is a News aggregator app which uses an API to fetch news from different sources and displays them in a unified way for users.

Signing up allows you to subscribe to News sources and lets you access Discover page where you can read all the news from your subscribed sources.

You must be subscribed to edit your profile. In the top right corner there is a round button, which by default shows placeholder profile picture, that can be clicked to see dropdown menu for going to profile page.

You can subscribe to News sources by first signing up and then navigating to Sources tab or any other news tab where you will come across a list of sources and a subscribe(plus) button for each Source.

You can unsubscribe from News sources by navigating to Sources tab or any other news tab where you will see sources, which you have subscribed to, with a minus sign which you can click to unsubscribe to that source.
